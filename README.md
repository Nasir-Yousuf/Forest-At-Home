🌱 Forest at Home

Forest at Home is a modern, interactive e-commerce web application for buying indoor plants.
The project focuses on creating a visually immersive, nature-inspired experience using animations, clean UI, and scalable frontend architecture.

🚀 Vision

The goal of this project is not just to build an online store, but to create a living digital environment — where users feel like they are exploring a growing forest rather than browsing a typical website.

Planned features include:
• 🌿 Animated homepage (growing trees, flowing leaves, organic motion)
• 🪴 Indoor plant catalog with detailed product pages
• 🛒 Shopping cart and checkout system
• ❤️ Wishlist functionality
• 🔍 Smart search and filtering
• 📱 Fully responsive design
